What is e-o?

e-o is just like any other browser-based visitor you have. Except this visitor is scripted and told to report back if certain things are missing or changed. These things can be numerous things:

  • A status code other than 200.
  • A missing background image.
  • A 3rd party plugin that stopped working.
  • A JavaScript syntax error or a console.log that was forgotten.
  • A webfont that stopped working.
  • A heading that no longer exists.
  • A missing css file.
  • An advertisement that changed or stopped showing.
  • A google analytics event that stopped being sent.
  • ...and so on.

If any of these (configurable) options returns something unexpected, I get an email with all the juicy debug info. And a screenshot.

So now I know, that everything is not 200 OK.
